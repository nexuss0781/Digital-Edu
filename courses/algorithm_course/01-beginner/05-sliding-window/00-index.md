# Sliding Window

## What You Will Learn

- Fixed vs dynamic sliding window
- Window expansion and shrinking technique
- Using hash maps inside windows for character counting
- Subarray/substring optimization from O(n²) to O(n)
- When to use sliding window vs two pointers

## Why This Matters

Sliding window solves contiguous subarray/substring problems efficiently. Instead of recalculating from scratch for every subarray, maintain a running window that slides forward — adding one element and removing one element per step.
