# Basic Recursion

## What You Will Learn

- What recursion is: a function calling itself
- Base case and recursive case
- The call stack and how recursion uses memory
- Tail recursion vs tree recursion
- Converting recursion to iteration
- Simple recursive problems (factorial, Fibonacci, power)

## Why This Matters

Recursion is the foundation of divide-and-conquer, backtracking, dynamic programming, and tree/graph traversal. Every advanced algorithm in this curriculum relies on understanding how recursion works — and when to convert it to iteration for efficiency.
