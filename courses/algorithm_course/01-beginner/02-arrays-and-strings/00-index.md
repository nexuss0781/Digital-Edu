# Arrays and Strings

## What You Will Learn

- How arrays work in memory (contiguous allocation, O(1) access)
- Array operations: insertion, deletion, traversal, two-pass techniques
- String manipulation: characters, substrings, immutable vs mutable
- Edge cases: empty arrays, single elements, duplicates
- In-place operations and space optimization

## Why This Matters

Arrays are the most fundamental data structure. Nearly every algorithm problem involves arrays in some form. Understanding array manipulation — index arithmetic, in-place swaps, boundary checks — is prerequisite to every pattern you will learn.
