# System Design

## What You Will Learn

- Distributed systems fundamentals (rate limiting, caching, load balancing)
- Cloud and infrastructure architecture
- AI/ML system design
- Real-world system designs (social media, e-commerce, streaming)
- Database design and data modeling
- Scalability, reliability, security, and observability

## Why This Matters

System design interviews test your ability to architect large-scale systems. This module provides reference material across 235 topics — from distributed rate limiters to AI-native applications. Use it as a reference, not a sequential study.

## Prerequisites

- All previous tiers (Beginner through Advanced)

## Estimated Time

Self-paced reference (not sequential study)
