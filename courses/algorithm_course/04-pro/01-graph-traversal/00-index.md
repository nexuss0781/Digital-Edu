# Graph Traversal

## What You Will Learn

- Graph representation: adjacency list vs adjacency matrix
- Depth-First Search (DFS): stack-based, recursive
- Breadth-First Search (BFS): queue-based, level-order
- Cycle detection in directed and undirected graphs
- Connected components and bipartite checking
- Topological sort for dependency ordering

## Why This Matters

Graphs model relationships — social networks, road maps, dependencies, networks. DFS and BFS are the two fundamental traversal patterns. Almost every graph problem starts with one of these two. Topological sort solves scheduling and dependency problems.
