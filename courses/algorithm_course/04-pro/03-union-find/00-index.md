# Union-Find (Disjoint Set)

## What You Will Learn

- Disjoint set data structure
- Union by rank and path compression
- O(alpha(n)) amortized union and find operations
- Applications: connectivity, cycle detection, component counting
- When Union-Find beats DFS/BFS

## Why This Matters

Union-Find efficiently tracks which elements belong to the same group. It solves connectivity problems faster than DFS/BFS — particularly when you need to process edges online or detect cycles dynamically.
