# Shortest Paths

## What You Will Learn

- BFS for unweighted graph shortest paths
- Dijkstra's algorithm for weighted graphs (non-negative)
- Bellman-Ford for graphs with negative weights
- Floyd-Warshall for all-pairs shortest paths
- When to use which algorithm

## Why This Matters

Shortest path algorithms power GPS navigation, network routing, and game pathfinding. Dijkstra's is the most common interview algorithm for weighted graphs. Understanding when negative weights break Dijkstra's is critical.
