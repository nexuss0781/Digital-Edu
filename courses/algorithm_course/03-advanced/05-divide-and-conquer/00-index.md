# Divide and Conquer

## What You Will Learn

- Split, solve, combine strategy
- Merge sort and quicksort as canonical examples
- Tree recursion: when subproblems form a tree
- Recurrence relations and Master Theorem basics
- When divide and conquer applies vs DP

## Why This Matters

Divide and conquer breaks problems into independent subproblems, solves each recursively, then combines results. It powers sorting algorithms, binary search, and many tree algorithms. Understanding it is essential for analyzing algorithm complexity.
