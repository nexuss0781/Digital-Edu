# Dynamic Programming — 2D

## What You Will Learn

- Grid DP: navigating 2D matrices with state transitions
- String DP: comparing two strings (LCS, edit distance)
- Interval DP: problems on subarrays/substrings
- State transitions on two dimensions
- Space optimization techniques

## Why This Matters

2D DP extends 1D thinking to problems with two changing parameters — comparing two strings, navigating a grid, or partitioning intervals. These problems appear frequently in interviews and require careful state definition.
