# Dynamic Programming — 1D

## What You Will Learn

- Overlapping subproblems and optimal substructure
- Memoization (top-down) vs tabulation (bottom-up)
- State definition: what changes between subproblems
- 1D DP patterns: Fibonacci, climbing stairs, house robber, coin change
- Converting recursion to iterative DP

## Why This Matters

Dynamic programming is the most important algorithm paradigm. It transforms exponential recursion into polynomial solutions by solving each subproblem once and storing the result. 1D DP is where you learn to think about state and transitions.
