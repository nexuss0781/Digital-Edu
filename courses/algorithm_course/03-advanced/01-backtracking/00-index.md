# Backtracking

## What You Will Learn

- Decision trees and state spaces
- Build candidates incrementally, abandon failures
- Permutations, combinations, subsets
- Constraint satisfaction (N-Queens, Sudoku)
- The "undo" technique: restore state after exploring a branch
- Pruning: cutting off unnecessary search branches

## Why This Matters

Backtracking systematically explores all possible solutions by building candidates step-by-step and backing up when a candidate fails. It solves combinatorial problems that brute force handles poorly — and it is the foundation for understanding DFS on graphs.
