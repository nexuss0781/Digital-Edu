# Stacks and Queues

## What You Will Learn

- Stack (LIFO): push, pop, peek, bracket matching
- Queue (FIFO): enqueue, dequeue, BFS traversal
- Monotonic stack: next greater/smaller element pattern
- Using stacks for DFS and expression evaluation
- Deque for sliding window maximum

## Why This Matters

Stacks and queues are the building blocks of tree traversal (DFS uses stack, BFS uses queue), expression parsing, and the monotonic stack pattern — which solves an entire class of "next greater element" problems in O(n).
