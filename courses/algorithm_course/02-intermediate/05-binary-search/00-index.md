# Binary Search

## What You Will Learn

- Search space reduction on sorted data
- Standard binary search template (left, right, mid)
- Modified binary search: first/last occurrence, search in rotated arrays
- Binary search on answer space (when the answer is numeric)
- Boundary handling and off-by-one errors

## Why This Matters

Binary search achieves O(log n) by eliminating half the search space each step. Beyond sorted arrays, it applies to "search on answer" problems — when you can verify a candidate answer in O(n), binary search finds the optimal answer in O(n log answer).
