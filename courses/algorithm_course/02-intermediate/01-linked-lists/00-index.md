# Linked Lists

## What You Will Learn

- Singly linked lists, doubly linked lists, circular linked lists
- Traversal with dummy nodes to handle edge cases
- Fast-slow pointer technique (cycle detection, middle finding)
- Reversal patterns (iterative, recursive)
- When linked lists outperform arrays

## Why This Matters

Linked lists teach pointer manipulation — the ability to rewire connections between nodes. This skill transfers to trees, graphs, and any data structure built from nodes. Cycle detection using fast-slow pointers is a classic interview pattern.
