# Heaps and Priority Queues

## What You Will Learn

- Min-heap and max-heap structure
- Priority queue abstraction using heaps
- Top K problems (Kth largest, K closest points)
- Finding median from a data stream (two heaps)
- Merging K sorted sequences

## Why This Matters

Heaps efficiently track the "maximum" or "minimum" element as data changes. They power priority queues, scheduling algorithms, and the two-heaps pattern for median finding. Many medium/hard problems require heap thinking.
