# Trees and Binary Search Trees

## What You Will Learn

- Tree terminology: root, leaf, height, depth, sibling
- Traversals: in-order, pre-order, post-order, level-order
- Binary Search Tree (BST) properties and operations
- Recursion on trees (natural fit for recursive thinking)
- Lowest Common Ancestor (LCA)
- When to use BFS vs DFS on trees

## Why This Matters

Trees appear everywhere: file systems, DOM, organization charts, decision trees, and ASTs. BSTs enable O(log n) search, insertion, and deletion when balanced. Tree traversal patterns are the foundation for graph traversal.
